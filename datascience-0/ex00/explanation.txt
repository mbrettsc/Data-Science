Exercise 00: Database Creation

Objective:
Create a PostgreSQL database named 'piscineds' as specified.

Requirements:
1. Create a new PostgreSQL database called 'piscineds'.
2. Ensure that the database can be accessed using the following credentials:
   - Username: your_login
   - Database Name: piscineds
   - Password: mysecretpassword
3. Test the connection to the database using the command:
   psql -U your_login -d piscineds -h localhost -W
   mysecretpassword

Instructions:
- Follow the instructions provided by your instructor or use PostgreSQL tools to create the database.
- Ensure the database setup is correct and verify connectivity.

Note:
No additional files need to be submitted for this exercise. The focus is on ensuring the database creation and connectivity are properly handled.
